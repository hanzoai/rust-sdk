pub struct DID {
    pub method: String,
    pub id: String,
    pub path: Option<String>,
    pub query: Option<String>,
    pub fragment: Option<String>,
}
Expand description

A W3C compliant Decentralized Identifier

Format: did:method:method-specific-id Example: did:hanzo:eth:0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb7

Fields§

§method: String

The DID method (e.g., “hanzo”, “ethr”, “key”, “web”)

§id: String

Method-specific identifier

§path: Option<String>

Optional path component

§query: Option<String>

Optional query parameters

§fragment: Option<String>

Optional fragment (for referencing parts of DID document)
